# AGENTS.md

This project uses **CLAUDE.md** as its primary instruction document for all AI coding agents.

If you are an AI agent (Claude Code, Codex, Gemini CLI, Cursor, Copilot, or other):

**→ Read [CLAUDE.md](CLAUDE.md) before writing any code.**

CLAUDE.md contains:
- What this project does and why
- The 4-stage pipeline architecture
- File-by-file module map
- CSV schema (the contract between stages)
- Code style rules
- Testing guidance
- Git workflow

CLAUDE.md is the single source of truth. AGENTS.md exists only to point non-Claude-Code agents here, since Claude Code reads CLAUDE.md natively while other runtimes look for AGENTS.md first.
