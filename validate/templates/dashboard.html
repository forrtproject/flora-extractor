🔬 Analysis — analysis/

Analysis Scripts & Outputs

  • gap_analysis.py Compares extracted.csv against the FLoRA entry sheet to find unextracted DOIs
  • rule_analysis.py Audits filter rules: contribution per rule, false-positive rate, improvement opportunities
  • apa_resolver.py Resolves APA-format citations in extracted rows to structured DOIs via CrossRef
  • extraction_audit.md Manual audit notes — accuracy estimate and common error patterns
  • gap_analysis_doi_matched.csv DOIs in FLoRA sheet matched to extracted.csv — gap between known and extracted
  • gap_analysis_url_matched.csv URL-matched gap analysis (fallback when DOIs differ by formatting)
  • gap_summary.md Human-readable summary of the gap analysis results

Run with: python -m analysis.gap_analysis    python -m analysis.rule_analysis    python -m analysis.apa_resolver
