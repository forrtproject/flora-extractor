📖 Analysis Scripts & Outputs

How to Run

  • run_overlap_analysis.py Main gap analysis — compares extracted.csv against FLoRA entry sheet to find unextracted DOIs. Writes gap CSVs and gap_summary.md.
  • rule_analysis.py Audits filter rules: contribution per rule, false-positive rate, improvement opportunities. Writes rule_improvement_opportunities.csv.
  • apa_resolver.py Resolves APA-format citations in extracted rows to structured DOIs via CrossRef. Writes apa_reference_fallback.csv.
python -m analysis.run_overlap_analysis python -m analysis.rule_analysis python -m analysis.apa_resolver

Output Files

  • gap_analysis_doi_matched.csv DOIs in FLoRA sheet matched to extracted.csv — full gap list with original and replication metadata
  • gap_analysis_url_matched.csv URL-matched gap analysis (fallback when DOIs differ by formatting)
  • gap_vs_extracted_comparison.csv Side-by-side comparison of FLoRA entries vs. what was extracted
  • rule_improvement_opportunities.csv Actionable suggestions from rule_analysis.py with priority ratings
  • gap_summary.md Human-readable executive summary of the gap analysis (displayed above)
  • extraction_audit.md Manual audit notes — accuracy estimate, link method distribution, missing DOI sample
